## Como pegar uma imagem do repositório

1. Abra a pasta onde a imagem está localizada no GitHub.

2. Encontre a imagem que deseja utilizar.

3. Copie o nome do arquivo.

Exemplo:

```
image%20298.png

```

# 🚨 IMPORTANTE - NOMES DE IMAGENS COM ESPAÇOS 🚨

> [!WARNING]
> Se o nome da imagem possuir espaços, **você DEVE substituir cada espaço por `%20` na URL**, caso contrário a imagem não será carregada.


4. Cole o nome da imagem no final do link abaixo:

```txt
https://raw.githubusercontent.com/luana-76/Imagens-ConexaoAprendiz/main/cabelo/image%20298.png
```

5. Utilize o link no código:

```jsx
<img
  src="https://raw.githubusercontent.com/luana-76/Imagens-ConexaoAprendiz/main/cabelo/image%20298.png"
  alt="Cabelo"
/>
```
